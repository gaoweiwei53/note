# Principle
1. 装系统使用 GPT + UEFI 方式，文件系统使用NTFS。装机时Bios禁用 secure boot 和 fast boot。打开quick boot，可加快每次开机速度。装机后可打开fast boot，但这只能加快win8和win10的开机速度。
2. `sudo passwd root`设置root密码
